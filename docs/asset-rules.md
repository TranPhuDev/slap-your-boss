# Asset Rules

Allowed:

- original SVG/Canvas/PixiJS assets;
- public-domain assets with source;
- licensed assets recorded in THIRD_PARTY_NOTICES.md.

Required groups:

```text
public/models
public/cursors
public/sounds
public/characters
src/assets/effects
```

Cursor assets:

- transparent SVG/PNG;
- ready and hit states;
- documented hotspot;
- desktop only.

No copyrighted commercial audio clips. Optimize raster/audio sizes and preload only required assets.
